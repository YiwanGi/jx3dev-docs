# 调用API接口

# 请求仅支持 POST 方法，通过 JSON 传递参数；目前服务器仅支持通过 JSON 格式返回数据。
  • 请求头中的 Dev-Token 为你的 16位 devToken
  • 请求头中的 Content-Type 必须对应的为 application/json
  • 你必须提供 ts 参数且与服务器差值应为 ±5 分钟，否则将被视为非法请求。
# 调用 数据接口 的 index 接口 <获取API列表>
POST /api/index HTTP/1.1
Host: data.jx3dev.com
Dev-Token: {{devToken}}
Content-Type: application/json

{
    "ts": 1697984542
}

# 请求成功示例

{
    "code": 200,
    "result": {
        "data": {
            "controller": "api",
            "interface": [...]
        },
        "extra": {
            "ts": 1697984542
        }
    },
    "msg": "ok"
}

# 请求失败示例

{
    "code": 200,
    "result": {
        "errorCode": -10000,
        "errorExtra": {
            "ts": 1697984542
        }
    },
    "msg": "Unknown error"
}
