# sms_captcha_send

# 发送验证码

# 参数

字段名数据类型默认值说明必须
mobileint(11)-手机号√
eventstring-事件名称√
# event 字段支持以下事件值
事件值
用户注册register
重置密码resetpwd

# 响应数据

字段名数据类型说明
statusenum(-2,-1,0,1)状态
# status 字段负值说明
值说明
-2发送受到限制
-1发送过于频繁
