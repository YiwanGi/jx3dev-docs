# 短信指令

注意

回复任意72小时内来自 JX3DEV 的短信即可实现以下操作。
特别说明: 回复后系统自动处理指令请求，无回执信息。

# BDXXXXX 绑定QQ

XXXXX为你要绑定的QQ号码(5-10位数)，仅支持QQ原始号码。
一个QQ支持绑定多个用户账号，用户账号使用权归属于绑定QQ所有。
绑定QQ可以联系管理员进行(包括但不限于)重置密码，冻结、解冻账号等操作。

# HQLETWS 获取ws使用许可

回复后自动取得ws使用许可。

# LQVIPT7 领取7天VIP体验时长

每个用户每年可领取一次。
